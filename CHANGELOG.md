## [1.0.0] - 2025-05-11

### Added
- Initial release of AWS Organization SCP module
- Support for creating and managing Service Control Policies
- Support for attaching SCPs to organization roots, OUs, and accounts
- Dynamic policy document generation with variable substitution
- Comprehensive tagging support following PC-IAC-004 standards
- Example implementation in sample/ directory
